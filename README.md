# Урок 1: Основы Git

Добро пожаловать на первый урок! Здесь вы найдёте всю теорию и практические материалы по основам системы контроля версий Git.

## Содержание

*   **Теория:** [`Теория.html`](Теория.html)  
    Полное руководство по ключевым понятиям: что такое Git, три состояния файла (modified → staged → committed) и основные команды.
*   **Практика:** Задание "Текстовый бот Вика" (описано в теории).
*   **Проверка знаний:** [`Тестирование.html`](Тестирование.html)  
    Интерактивный тест для проверки вашего понимания пройденных команд.

## Пройденные команды

На этом уроке вы изучили следующие команды:
*   `git config --global user.name` и `user.email`
*   `git init`
*   `git status`
*   `git add <file>` и `git add .`
*   `git commit -m "сообщение"`
*   `git log --oneline`

Удачи в освоении!

# Урок 3: Основы pygame-ce

Добро пожаловать на третий урок! Здесь вы найдёте теорию и практические материалы по созданию 2D-игр с помощью библиотеки **pygame-ce**.

## Содержание

*   **Теория:** [`Теория.html`](Теория.html)  
    Полное руководство по ключевым концепциям: что такое pygame-ce, структура игрового цикла (обработка событий → обновление → отрисовка), основные компоненты (`Window`, `Surface`, `Rect`, `Clock`, `Vector2`) и важные принципы работы.
*   **Практика:** Проект "Tower Defense" — мини-игра, где вы управляете башней и защищаетесь от врагов.
*   **Проверка знаний:** [`Тестирование.html`](Тестирование.html)  
    Интерактивный тест для проверки вашего понимания основ pygame-ce и игрового цикла.

## Пройденные концепции

На этом уроке вы изучили следующие ключевые идеи:
*   Структура игрового цикла.
*   Создание окна игры с помощью `pygame.Window`.
*   Использование `pygame.Clock` для контроля FPS.
*   Работа с объектами `Surface` и `Rect`.
*   Применение `pygame.Vector2` для расчёта движения.
*   Важные принципы: порядок отрисовки, очистка экрана, обновление дисплея.

# Курс по разработке игр на Python

Теоретические материалы и практические задания для студентов.

## Список уроков

| №   | Тема                | Описание                                                                                                                               |
| --- | ------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| 1   | **Основы Git**      | Введение в систему контроля версий. Рабочий процесс: modified → staged → committed. Базовые команды `init`, `add`, `commit`, `status`. |
| 2   | **GitHub и Markdown** | Работа с удалёнными репозиториями: команда `git push`, создание аккаунта на GitHub. Оформление проектов с помощью файла `README.md` и языка разметки Markdown. |
| 3   | **Основы pygame-ce** | Знакомство с библиотекой для создания 2D-игр. Структура игрового цикла, работа с окном, спрайтами, событиями и движением. |

> **Как пользоваться:** Чтобы открыть материал урока, перейдите в соответствующую ветку репозитория.
